🪪 License Management ERP System
📌 Project Overview

This project is an ERP-style web application designed to manage licenses, track related statistics, and control access through an authentication system.

It includes user login functionality, protected dashboard views, and backend logic to manage license data and operational insights.

🎯 Purpose

To centralize and manage license-related data through a secure dashboard interface while providing statistical insights for monitoring and decision-making.
